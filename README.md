# Angular Banking Transaction System
Frontend banking transaction management application built with Angular & TypeScript.
Simulates a bank account with transaction validation, balance tracking, and persistent encrypted storage.

## Features
- Add and validate transactions (date, amount, type)
- Automatic balance calculation
- Prevent invalid transactions
- Encrypted LocalStorage persistence
- Dynamic transaction table
- Conditional rendering with *ngIf
- Iteration with *ngFor
- Delete transactions functionality
- Israeli date & currency formatting using Angular Pipes
- Conditional row styling (negative balance highlight)

## Technologies
- Angular
- TypeScript
- HTML
- CSS
- LocalStorage API
  
## Run Locally
- npm install
- ng serve

Open: http://localhost:4200
